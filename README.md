# Desentupidora Nuclear — Landing Page

Site institucional responsivo construído com **TanStack Start + Vite + React 19 + Tailwind CSS v4**.

## 🚀 Rodando no StackBlitz

1. Acesse [stackblitz.com](https://stackblitz.com/) e clique em **Create project → Import from GitHub** (ou faça upload do projeto).
2. O StackBlitz detecta o `.stackblitzrc` e roda automaticamente:
   ```bash
   npm install
   npm run dev
   ```
3. O preview abre em uma URL pública pronta para compartilhar.

> Requer Node.js 20+ (já configurado em `engines`).

## 💻 Rodando localmente

```bash
npm install
npm run dev      # ambiente de desenvolvimento
npm run build    # build de produção
npm run preview  # preview do build
```

## 📦 Stack

- TanStack Start (SSR + file-based routing)
- React 19 + TypeScript
- Tailwind CSS v4
- shadcn/ui (Radix)
- Lucide Icons
